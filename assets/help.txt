Archon Backup Utility
----------------------------------------
Archon
Ussage: python3 __main__.py command

Commands:
    fbackup     - Backup entire folders
    frestore    - Restore folder from Backup
    fbackupftp  - Backup folder to FTP(S) server
    frestoreftp - Restore folder from FTP(S) server
    history     - Read backup history
    help        - Show this help
